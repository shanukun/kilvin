Getting Started

A simple static site generator is Kilvin. It takes markdown text and turns it into a static webpage using layouts. Changes can be made to the page's content, URLs, and the way the site looks.

Prerequisites

Kilvin requires the following:

  • Python version 3.9 or higher
  • pip : package installer

Instructions

  1. Install all prerequisites.
  2. Install the kilvin.
pip install kilvin

Create a site

  1. Create a new kilvin site at ./my_project
kilvin init my_project
  1. Change into your new directory.
cd my_project
  1. Build the site.
kilvin build
  1. Make it available on local server.
kilvin server
